I am creating this short readme for your beneift. To start this app, run npm start in your terminal and your browser will open it automatically.
Enter a valid search in the input field and hit submit to display the search results
Use the reset button to reset the displayed data to the beginnning data.
Use the Load more button to increase the display of results by a count of 10
Use the Load less button to decrease the display of results by a count of 10
use the Sort ascending name button to sort the data by ascending name
use the Sort descending name button to sort the data by descending name
Click on the Brewery title to be routed to the brewery detail page.
I have completed the required exerecises (one and two), as well as completeing extra exercises (styling, pagination, and sorting by name descending or ascending).
